// Case Studies

Uitgelichte projecten met meetbare impact.

map
Geo-visualisatie

GeoInzicht Platform

Een compleet data warehouse en dashboard dat 16 CBS-domeinen integreert: van bevolkingssamenstelling en inkomensverdeling tot energieverbruik en woningvoorraad. Alles zichtbaar op buurtniveau via interactieve kaarten.

  • 16 datadomeinen op 1 kaartdashboard
  • PostGIS + GeoServer + Leaflet stack
  • Health monitoring per domein
  • ETL-pipelines voor CBS open data
shield
Privacy & compliance

Meldpunt Ambtenaren

Een Progressive Web App voor ambtenaren om misstanden te melden. Documenten worden automatisch geanonimiseerd en dossiers veilig beheerd via Supabase met row-level security.

  • PWA met offline functionaliteit
  • Automatische document-anonimisering
  • EHRM-wizard voor begeleiding
  • Veilige opslag via Supabase RLS
music_note
Creatieve technologie

Browser DAW

Een volledige digitale audio werkplaats in de browser. Timeline-editor, mixer, virtuele instrumenten en een sound library. Gebouwd met de Web Audio API voor professionele geluidskwaliteit.

  • Multi-track timeline met drag & drop
  • Web Audio API synthesizers
  • Freesound sample library integratie
  • PostgreSQL project opslag

// Over

DataConsultants bouwt dataprojecten die er toe doen. Van CBS-buurtanalyses die lokaal beleid ondersteunen tot creatieve tools die de wereld een beetje leuker maken.

Onze missie is simpel: data inzetten voor een betere wereld — en daar plezier in hebben. Elk project begint met nieuwsgierigheid en eindigt met werkende software.

We werken met open data, open source en moderne webtechnologie. Geen complexe frameworks, maar snelle, efficiente applicaties die gewoon werken.
