GeoInzicht DWH

Data warehouse voor CBS buurtdata. Verzamelt, modelleert en ontsluit open data uit 6+ bronnen via een gestandaardiseerd stermodel.

Actief

info Wat is GeoInzicht DWH?

GeoInzicht DWH is het data warehouse achter het GeoInzicht platform. Het verzamelt CBS buurtdata, BAG adressen, criminaliteitsstatistieken, zorgkosten, flora- en faunagegevens en meer in een centraal PostgreSQL/PostGIS stermodel.

De data wordt opgehaald uit diverse open databronnen, getransformeerd via een staging-laag in SQL Server, en vervolgens geladen in een gemodelleerd data warehouse met feiten- en dimensietabellen. Dit maakt het mogelijk om data uit verschillende domeinen te combineren en te analyseren op buurt-, wijk- en gemeenteniveau.

Het DWH voedt de GeoInzicht App met schone, gemodelleerde data via een Flask REST API en GeoServer voor ruimtelijke services (WFS/WMS). Zo wordt ruwe open data omgezet in bruikbare inzichten.

architecture Architectuur

De data doorloopt een gestandaardiseerde ETL-pipeline van bron tot dashboard:

1

Brondata

CBS OData, BAG/Kadaster, Politie Open Data, Vektis Zorgkosten, GBIF Natuur, PDOK Geo

2

Staging

Ruwe data wordt opgehaald en opgeslagen in SQL Server staging tabellen voor validatie en deduplicatie.

3

DWH

Getransformeerde data wordt geladen in PostgreSQL/PostGIS met een stermodel (feiten + dimensies).

4

Geo Services

GeoServer publiceert ruimtelijke data als WFS/WMS services voor kaartvisualisaties.

5

Flask REST API

Python Flask API ontsluit de DWH-data via gestandaardiseerde endpoints voor het dashboard.

6

Dashboard / Leaflet

De GeoInzicht App visualiseert de data op een interactieve kaart met indicatoren en vergelijkingen.
